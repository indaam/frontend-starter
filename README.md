frontend-starter
================

Frontend starter use compass &amp; jade with grunt

How to use
-----------

  1 install node.js
  2 clone this repo
  3 Open cmd
  4 npm install -g grunt-cli
  5 npm install grunt --save-dev
  6 npm install
  7 grunt jalan
  8 end :D
